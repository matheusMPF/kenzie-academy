Conteúdo do curso

O que eu vou aprender neste curso?

Nesse curso você vai aprender na prática, direto da tela do meu computador como criar do zero um site completo.

Vamos te ensinar os princípios das três principais linguagens de programação do mercado de tecnologia:

  • HTML

    HTML organiza o seu site.

  • CSS

    CSS deixa o seu site mais bonito.

  • Javascript

    Javascript traz interatividade.

Com estas habilidades, você não só será capaz de criar páginas impressionantes, mas também abrirá portas para oportunidades no mundo da tecnologia!
